### nba.lineup_stats_season

- league_id text
- season_year integer
- season_label text
- season_type text
- team_id integer REFERENCES nba.teams(team_id)
- player_ids integer[] -- Array of nba_id (sorted)
- per_mode text
- measure_type text
- gp integer
- minutes numeric(8,2)
- off_rating numeric(5,2)
- def_rating numeric(5,2)
- net_rating numeric(5,2)
- ast_pct numeric(5,4)
- ast_tov numeric(5,2)
- ast_ratio numeric(5,2)
- oreb_pct numeric(5,4)
- dreb_pct numeric(5,4)
- reb_pct numeric(5,4)
- tm_tov_pct numeric(5,4)
- efg_pct numeric(5,4)
- ts_pct numeric(5,4)
- usg_pct numeric(5,4)
- pace numeric(6,2)
- pie numeric(5,4)

- created_at timestamptz
- updated_at timestamptz
- fetched_at timestamptz

#### Constraints & Indexes
- PRIMARY KEY (league_id, season_year, season_type, team_id, player_ids, per_mode, measure_type)
- INDEX ON (season_year, season_type)
- INDEX ON (team_id)
- INDEX ON (player_ids)
- INDEX ON (league_id)

#### Notes
- Sourced from Query Tool endpoints like `/season/lineups`.
- This table is season-aggregated lineup performance; it should not be mixed with game-level lineup splits.
- `player_ids` is a sorted array of official `nba_id`s representing the lineup.
